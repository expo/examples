## 🚀 How to use

- Install with `yarn` or `npm install`.
- Create your own app on [Magic.link](https://magic.link).
- Open `App.js` and replace `MAGIC_KEY` with your publishable key.
- Run `yarn start` or `npm run start` to try it out.
